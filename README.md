# makeDeb
a .deb skeleton

# Compilation du paquet:
```$ sudo dpkg-deb --build nom_du_paquet```

# Installation du paquet:
```$ sudo dpkg -i nom_du_paquet.deb```

# Desinstallation du paquet:
```$ sudo apt-get remove nom_du_paquet```
